The below table describes the steps in the admission application process as well as the visa application process

Step No. Step Category Responsible Actions
1 e-brochure Application Processing Student Student to go through brochures and access course information
2 TRUE application Form Application Processing Agent Agent gets application filled by student, Agent makes sure that all relevant steps and required documents (from 2-7 below) are provided.
3 IELTS/TOEFL or equivalent results Application Processing Student Student should submit IELTS/TOEFL results along with application to agent
4 Curriculum Vitae Application Processing Student Student has to prepare a CV and pass it on the agent along with the application form
5 Three Letters of Recommendation Application Processing Student Students can obtain letters of recommendation from their University or place of work
6 Motivation Letter/Statement of Purpose Application Processing Student Student needs to clearly mention the purpose of study. The student **must* clearly state in the letter that he/she will leave the country once the study period is completed.
7 Photocopy of the passport Application Processing Student Student needs to provide a photocopy of his/her recent valid passport and Get the copies attested by Justice of Peace/Notary as True copy of the original and submit it along with the application.
8 Interview with TRUE representative or TRUE agent Application Processing Agent Agent should liaison between student and TRUE staff for an interview to assess and screen for
  • Financial documents of student
  • Statement of Propose
  • All other relevant documents
    • 9 Letter of Offer Application Processing TRUE within two weeks from the date of receiving complete application Based on the interview with TRUE representative (Step 8), as an application outcome, Letter of Offer is issued by TRUE
    10 Initial Payment Application Processing Student
  • With the financial evidence (to cover the whole fees & living costs to study at University) are provided, documents from bank, the initial payment specified in the offer letter (step 9) should be made
  • The financial evidence (to cover the whole fees & living costs to study at University) from a bank must to be submitted to University through TRUE
    • 11 Letter of Acceptance and Enrolment Confirmation Application Processing TRUE When financial evidence and initial payment are done (Step 10), University through TRUE will issue letter of Acceptance and Enrolment.
    12 Apply for Visa Visa Application Agent
  • After receiving letter of Acceptance and Enrolment Confirmation (Step 11), student can apply for a visa (of his/home country).
  • NOTE: Visa requirements to enter are different from a country to another. Therefore the agent will need to carefully read the requirements and to thoroughly check the required students’ documents before submitting them to the respective Embassy. If some of information are not correct, the application could be rejected.
    • 13 Visa Application Form Visa Application Agent/Student Download and fill up the VISA application form from student’s home country listed in Mauritius Government website listed.
    14 3 (three) passport size photos Visa Application Student - It is mandatory that student submit 3 passport size photos
    15 3 (three) copies of passport Visa Application Agent/Student All photocopies must be attested by Justice of Peace in your home country or Notary.
    16 1 (one) original and 2 (two) copies of “letter of acceptance and Enrolment Confirmation Visa Application Agent/Student - While applying for VISA, the student needs to submit original Letter of Confirmation of Enrolment to VISA office.
    17 3 (three) copies proof of pre-paid tuition fees to TRUE Visa Application Agent/Student Download and fill up the VISA application form from student’s home country listed in Mauritius Government website listed.
    13 Visa Application Form Visa Application Agent/Student An official receipt issued by the University through TRUE must be submitted along with the VISA application.
    18 Visa Application 3 (three) copies proof of sufficient funds to cover costs of stay, such as bank statements, saving books, etc (at least for the period of one year). Student This is VERY IMPORTANT. Proof of sufficient funds to pay the whole TRUE fees (for the whole duration of study) as well as the living expenditure for the whole duration of study) must be submitted along with the VISA application.
    19 3 (three) copies of degree/study certificates Visa Application Student Attested copies of degree certificates must be submitted along with the VISA application
    20 Curriculum Vitae Visa Application Student A brief CV with 2 referees must be submitted along with the VISA application
    21 IELTS/TOEFL certificate Visa Application Student IELTS/TOEFL certificate must be submitted along with the VISA application
